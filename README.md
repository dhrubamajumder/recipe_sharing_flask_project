# 🍲 Recipe Sharing Platform

A Recipe Sharing Platform built with Django and Django REST Framework where users can create, manage, search, and like recipes.

---

# 🚀 Features

## 👤 User Authentication
- User Registration
- Login / Logout
- User Profile Management

---

# 🍽 Recipe Features

- Create Recipe
- Read Recipe
- Update Recipe
- Delete Recipe
- Recipe Details Page
- Upload Recipe Images
- Recipe Categories

---

# ❤️ Like System

Users can:
- Like Recipes
- Unlike Recipes

---

# 🔍 Search System

Users can:
- Search Recipes by Name
- Search by Category

---

# 📖 Recipe Details

Each recipe includes:
- Recipe Title
- Ingredients List
- Cooking Instructions
- Cooking Time
- Recipe Image
- Author Information

---

# 👤 User Dashboard

Users can:
- Manage Their Recipes
- Edit/Delete Recipes
- View Liked Recipes
- Update Profile

---

# 📊 Reports & Analytics

- Total Recipes
- Most Liked Recipes
- Active Users
- Recipe Statistics

---
